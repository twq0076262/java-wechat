# 微信开发准备(一)--Maven 仓库管理新建 WEB 项目

在我们的项目开发中经常会遇到项目周期很长，项目依赖 jar 包特别多的情况，所以我们经常会在项目中引入 Maven 插件，建立起 Maven 项目，今天我就记录一个简单的 Maven 项目建立的简单流程！
